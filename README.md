# Food Order App

# 프로젝트 개요
장바구니에 상품을 추가하고 주문하는 앱 입니다.
배송지정보를 입력하면 주문된 상품들과 배송지 정보는 파이어베이스에 저장됩니다.
(결제기능 모듈은 연동하지 않았습니다)

# 기술스택
Javascript, CSS, React

# 핵심 구현기술
React - Context Api(데이터 스토어) 

# 배포(파이어베이스)
> https://react-order-app-86271.web.app
